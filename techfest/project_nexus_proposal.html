EVENT PROPOSAL: PROJECT NEXUS
(TechFest 2026 - Organizing Committee)
Theme: Gamifying the Future Date: 03-02-2026
1. EXECUTIVE SUMMARY
"Project Nexus" is a proposal to transform the standard technical exhibition into a fully interactive Cyber-Arcade. By integrating a centralized "FestCoin" economy, we aim to increase student engagement and participation across all 27 projects.
2. THE CORE MECHANIC: "FESTCOIN"
To gamify the event, every interaction is tracked digitally:
  • Entry: Every student receives a QR-Code Badge (User ID).
  • Play-to-Earn: Winning games (e.g., Deadshot.io) earns "FestCoins".
  • Leaderboard: A live dashboard projects the "Top 10 Players".
  • Redemption: Coins can be spent at the IoT Smart Dispenser for drinks/snacks.
3. EVENT ZONES & HIGHLIGHTS
Zone A: The Arena (Competitive Gaming)
High-energy competitions including the Deadshot.io Tournament (1v1 FPS) and Nose-Nav Flappy Bird (Facial recognition gaming).
Zone B: The Cyber-Lab (Security)
Demonstrations of cybersecurity concepts: The Phishing Tank (identifying fake sites), Password Strength Meter (entropy education), and basic ethical hacking demos.
Zone C: The AI Studio (Generative)
Creative stations where students use AI: AI Artist (Generative Art stickers) and Sonic Identity (Music generation from user profiles).
Zone D: Dev Sprints (Coding & IoT)
Technical challenges including 15-Minute Code Sprints and the IoT Smart Hydration system acting as the coin redemption point.
4. REQUIREMENTS
We require approval for:
  • Allocating 1 Lab (Computer Center) for the event.
  • High-speed internet access for API usage.
  • Permission to set up the IoT hardware (Water Pump/Relay).
