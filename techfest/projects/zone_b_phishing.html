Project: The Phishing Tank

Zone B: The Cyber-Lab

1. Overview

A cybersecurity awareness challenge. Participants are presented with real-world scenarios (Emails, Login Pages, WiFi Networks) and must identify which are legitimate and which are traps.

2. Gameplay & Rules

  • Challenge 1 - Email: Spot the fake sender address or malicious link.
  • Challenge 2 - Web: Identify the cloned Facebook/Google login page (Typosquatting).
  • Challenge 3 - WiFi: Which network is the "Evil Twin"?
  • Coin Reward: 20 FestCoins for 3/3 correct answers.

3. Technical Implementation

  • Offline Website: A locally hosted HTML quiz interface.
  • Evil Twin Demo: Use ESP8266 Deauther (Demonstration ONLY, secure environment).
  • Gophish: Use Gophish framework screenshots for educational examples.

4. Requirements

  • 1 Laptop for Quiz.
  • Educational posters explaining "HTTPS", "Sender Domain", etc.
