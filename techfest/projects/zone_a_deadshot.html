Project: Deadshot.io Tournament

Zone A: The Arena

1. Overview

A high-octane First Person Shooter (FPS) tournament hosted in the browser using Deadshot.io. This event serves as the main crowd-puller for the festival.

2. Gameplay & Rules

  • Format: 1v1 Deathmatch.
  • Duration: 3 minutes per round.
  • Coin Reward:
    • Winner: 50 FestCoins
    • Loser: 10 FestCoins (Participation)
    • Bonus: Defeat the "Sentinel AI" Bot: 100 FestCoins

3. Technical Setup

  • Platform: PCs running Chrome/Brave browser.
  • Network: Low latency ethernet connection required.
  • Sentinel AI: If implementing the bot challenge, a separate server running a computer vision aim-bot (OpenCV/YOLO) will play as the opponent.

4. Requirements

  • 4 Gaming PCs (Mouse, Keyboard, 144Hz Monitor preferred).
  • Stable Internet Connection.
  • Spectator Screen (Projector) for audience view.

Internal Notes: Ensure account creation is pre-handled or guests allow "Guest Mode".
