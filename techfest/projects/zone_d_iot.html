Project Details: IoT Smart Hydration
Zone: D (Dev Sprints) ID: D-02
1. Overview
A functional Real-World IoT implementation where students spend digital "FestCoins" to dispense physical drinks (Water/Juice).
2. Workflow
  1. User scans QR on standard Water Cooler.
  2. App confirms 10 Coin payment via Cloud API.
  3. Server sends MQTT payload to ESP32.
  4. ESP32 triggers Relay to turn on Pump for 5s (200ml).
3. Hardware Stack
  • Controller: ESP32 (WiFi + MQTT).
  • Actuator: 5V Relay + Submersible Pump.
  • Power: 12V DC Supply.
