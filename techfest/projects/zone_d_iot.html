PROJECT TECHNICAL SHEET: IoT SMART HYDRATION
Zone: D (Dev Sprints)
Project ID: D-02
Hardware: ESP32 / Arduino
Date: 03-02-2026
1. Project Overview
A functional IoT prototype demonstrating the "FestCoin" economy in the real world. Users scan a QR code at the water cooler, which deducts 10 FestCoins from their cloud balance and triggers a hardware relay to dispense 200ml of water.
2. User Flow
  1. User approaches the "Smart Cooler".
  2. Scans the static QR code on the machine using the FestApp.
  3. Confirms payment (10 Coins).
  4. App sends HTTP POST request to the server.
  5. Server validates balance and sends MQTT message to the Water Cooler.
  6. Pump activates for 5 seconds (calibrated for 200ml).
3. Hardware Stack
  • Microcontroller: ESP32 (Wi-Fi enabled).
  • Actuator: 5V/12V Relay Module controlling a Submersible Water Pump.
  • Power: 12V DC Adapter.
  • Display: 16x2 LCD showing "Ready" or "Dispensing...".
4. Protocol
  • Communication: MQTT (Message Queuing Telemetry Transport).
  • Broker: Mosquitto (Local) or HiveMQ (Cloud).
