Project Details: Password Strength Meter
Zone: B (Cyber-Lab) ID: B-02
1. Overview
An educational tool that calculates password entropy and estimates the time required to crack it using brute-force methods.
2. Logic
  • Entropy (E): Length * log2(Character Set).
  • Time to Crack: (2^E) / Guesses Per Second.
  • Library: zxcvbn (Dropbox) or custom Python script.
3. Privacy Notice
Strict Rule: No inputs are logged or saved. The system runs locally in RAM associated with the active session only.
