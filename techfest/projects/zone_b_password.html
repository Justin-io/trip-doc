Project: Password Strength Meter

Zone B: The Cyber-Lab

1. Overview

An interactive tool that educates users on entropy and brute-force attacks. Users type a password, and the system calculates how long it would take a hacker to crack it using current GPU capabilities.

2. Features

  • Real-time Feedback: "Time to Crack: 300 Centuries" vs "Time to Crack: 2 Seconds".
  • Suggestions: "Add a special character!" or "Make it longer!"
  • Coin Reward: 25 FestCoins for entering a password > 100 years crack time.

3. Technical Implementation

  • Library: zxcvbn (Dropbox's estimation library) or a custom Python entropy script.
  • Metrics: Length, Complexity (Upper/Lower/Symbol), Dictionary Words.
  • Computation Formula: Entropy E = L * log2(R). Time = (2^E) / Guesses_Per_Second.

4. Requirements

  • Standalone Laptop/Tablet.
  • Privacy Note: System must NOT log/save any passwords entered.
