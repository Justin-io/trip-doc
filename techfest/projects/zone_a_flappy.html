Project Details: Nose-Nav Flappy Bird
Zone: A (The Arena) ID: A-02
1. Overview
An interactive game where players control the bird using nose movements tracked via webcam, eliminating the need for a keyboard or mouse.
2. Mechanics
  • Input: Facial Landmarks (Nose Tip Y-Coordinate).
  • Action: Rapid upward movement triggers "Jump".
  • Tech: Python (OpenCV + MediaPipe) or JavaScript (p5.js + ml5.js).
