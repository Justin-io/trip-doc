PROJECT SRS & WORKSHOP: NOSE-NAV FLAPPY BIRD
Domain: Computer Vision / Game Dev | Duration: 45 Mins (Build-Along)
1. System Overview
This project is a hands-free interactive game where the player controls the "bird" using the vertical movement of their nose tip, tracked via a webcam in real-time. Students will learn to map Face Landmarks to Game Physics.
2. Workshop Agenda (45 Minutes)
Time Activity & Steps
0:00 - 10:00 Concept & Setup
- Theory: How Computers "See" (Pixels -> Grayscale -> Features).
- Tool: Intro to MediaPipe (468 Face Landmarks). Why Landmark #4 (Nose Tip)?
- Setup: `pip install opencv-python mediapipe pygame`
10:00 - 20:00 Step 1: The Vision (OpenCV + MediaPipe)
- Write script to open Webcam (`cv2.VideoCapture`).
- Apply MediaPipe FaceMesh.
- Goal: Draw a green dot on the user's nose in real-time.
20:00 - 35:00 Step 2: The Logic (Mapping Coordinates)
- Problem: Webcam is 640x480, Game is 800x600.
- Math: Normalization. `Game_Y = Nose_Y * Screen_Height`.
- Physics: Moving a rectangle based on this value.
35:00 - 45:00 Step 3: The Game (Collision & Polish)
- Add Pipes (moving left).
- Check Collision: `if Bird.rect.colliderect(Pipe.rect): Game Over`.
- Challenge: Add "Squat to Restart" feature.
3. Technology Stack
  • Language: Python 3.9+
  • Libraries: MediaPipe (Tracking), OpenCV (Input), PyGame (Rendering).
4. Core Code Snippet (MVP)
The core tracking logic students must write:
import cv2
import mediapipe as mp

# Initialize Face Mesh
mp_face_mesh = mp.solutions.face_mesh
face_mesh = mp_face_mesh.FaceMesh(max_num_faces=1)

cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()
    rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    results = face_mesh.process(rgb_frame)
    
    if results.multi_face_landmarks:
        for landmarks in results.multi_face_landmarks:
            # Landmark 4 is the Nose Tip
            nose = landmarks.landmark[4]
            h, w, _ = frame.shape
            cx, cy = int(nose.x * w), int(nose.y * h)
            
            # This 'cy' value controls the bird!
            cv2.circle(frame, (cx, cy), 15, (0, 255, 0), -1)

    cv2.imshow("Nose Tracker", frame)
    if cv2.waitKey(1) & 0xFF == ord('q'): break
