Project SRS: AI Artist & Meme Maker
1. System Overview
A kiosk application that allows users to type a text prompt and receive a unique, AI-generated image. This project demonstrates Prompt Engineering and API Integration with Generative Models.
2. Technology Stack
  • Frontend: HTML5 + Bootstrap (or React) for a clean UI.
  • Backend: Python Flask (to hide API keys).
  • AI Service: OpenAI DALL-E 3 API or Stability.ai API.
  • Storage: Local folder or Cloudinary (for QR code hosting).
3. Workflow Logic
  1. Input: User types "Cyberpunk cat eating pizza".
  2. Request: Frontend sends POST to Flask backend `[POST] /generate`.
  3. Generation: Backend calls `openai.Image.create`.
  4. Receive: OpenAI returns an Image URL.
  5. Display: Frontend renders the image `<img src="...">`.
4. Student Developer Guide (MVP)
Step 1: Setup Backend
from flask import Flask, request, jsonify
import openai

app = Flask(__name__)
openai.api_key = "YOUR_KEY"

@app.route('/generate', methods=['POST'])
def generate():
    prompt = request.json.get('prompt')
    response = openai.Image.create(
        prompt=prompt,
        n=1,
        size="512x512"
    )
    return jsonify({'url': response['data'][0]['url']})

if __name__ == '__main__':
    app.run(port=5000)
Step 2: Frontend JS
Make a Fetch call to your local server.
async function makeArt() {
    const txt = document.getElementById("prompt").value;
    const res = await fetch("http://localhost:5000/generate", {
        method: "POST",
        headers: {"Content-Type": "application/json"},
        body: JSON.stringify({prompt: txt})
    });
    const data = await res.json();
    document.getElementById("result").src = data.url;
}
