PROJECT SRS & WORKSHOP: AI ARTIST API
Domain: Generative AI / Web Dev | Duration: 45 Mins (Build-Along)
1. System Overview
A kiosk application where users generate custom images via text prompts. Students will build a robust backend to hide API keys and serve a simple "Text-to-Image" frontend using OpenAI's DALL-E or Stability AI.
2. Workshop Agenda (45 Minutes)

- Concept: Diffusion Models (Noise -> Image).

10:00 - 20:00 Step 1: The Backend (Flask Security)
- Problem: Never expose API keys in Frontend JS.

20:00 - 35:00 Step 2: The API Call (Connecting DALL-E)

35:00 - 45:00 Step 3: The Frontend (User Experience)

  • AI Provider: OpenAI API (DALL-E 3).
  • Frontend: HTML/JS (Fetch API).
4. Core Code Snippet (MVP)
The Python Backend Logic:
from flask import Flask, request, jsonify
import openai

app = Flask(__name__)
# NEVER hardcode this in production repository
openai.api_key = "sk-..." 

@app.route('/generate', methods=['POST'])
def generate_image():
    user_prompt = request.json.get('prompt')
    
    try:
        response = openai.Image.create(
            prompt=user_prompt,
            n=1,
            size="512x512"
        )
        image_url = response['data'][0]['url']
        return jsonify({'success': True, 'url': image_url})
        
    except Exception as e:
        return jsonify({'success': False, 'error': str(e)})

if __name__ == '__main__':
    app.run(debug=True, port=5000)
