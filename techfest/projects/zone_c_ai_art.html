Project Details: AI Artist & Meme Maker
Zone: C (AI Studio) ID: C-01
1. Overview
A generative art station where attendees input creative prompts to generate custom stickers or memes using Stable Diffusion or DALL-E.
2. User Flow
  1. User types prompt (e.g., "Cyberpunk cat").
  2. System calls AI API (OpenAI/Stability).
  3. Image generated in 15 seconds.
  4. QR Code displayed for download.
3. Requirements
  • High-Speed Internet (API Access).
  • Tablet for Input.
  • Secondary Monitor for Display.
