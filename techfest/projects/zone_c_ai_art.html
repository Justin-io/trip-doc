1. Project Overview
The "AI Artist & Meme Maker" is an interactive generative art station where attendees enter creative prompts to generate custom stickers or memes in real-time. The goal is to demonstrate the capabilities of multimodal AI models like Stable Diffusion or DALL-E in a fun, accessible way.
2. User Interactions
  1. Prompt Entry: User types a prompt (e.g., "Cyberpunk cat eating pizza") on a tablet.
  2. Generation: The system processes the request using a generative API (e.g., Stability AI or OpenAI).
  3. Output: A high-quality image is displayed within 10-15 seconds.
  4. Share: A QR code is generated for the user to download the image.
3. Technical Architecture
  • API Provider: OpenAI DALL-E 3 API or Stability AI (SDXL).
  • Backend: Python Flask/FastAPI server to handle requests and API keys securely.
  • Frontend: A simple React or HTML5 interface running on a kiosk mode tablet.
  • Display: Results mirrored to a secondary monitor for crowd visibility.
4. Hardware Requirements
  • 1x High-Performance Laptop (for hosting backend/mirroring).
  • 1x Tablet (iPad/Android) for user input.
  • Stable Internet Connection (Minimum 100 Mbps uplink).
