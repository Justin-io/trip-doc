EVENT RULES: 15-MINUTE CODE SPRINT
Zone: D (Dev Sprints)
Event ID: D-01
Type: Competitive Programming
Date: 03-02-2026
1. Objective
Participants must solve a single algorithmic problem within a strict 15-minute time limit. The focus is on logic, speed, and clean implementation.
2. Rules & Regulations
  1. Time Limit: Strictly 15 minutes per round. Timer starts when the problem is revealed.
  2. Languages: C++, Python, Java, or JavaScript are allowed.
  3. Resources:
    • Offline Documentation Allowed.
    • No Internet access (StackOverflow/ChatGPT/Copilot BANNED).
  4. Submission: Code must compile and pass 3 hidden test cases.
3. Scoring Criteria
  • Correctness (60%): Passes all test cases.
  • Time (30%): Faster submission = Higher score.
  • Style (10%): Variable naming and readability (Tie-breaker).
4. Infrastructure
  • Platform: HackerRank Contest (Private) or Local Judge (DOMjudge).
  • Hardware: 10 Lab PCs pre-installed with VS Code and compilers.
