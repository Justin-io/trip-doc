Event Rules: 15-Minute Code Sprint
Zone: D (Dev Sprints) ID: D-01
1. Objective
Solve a single algorithmic problem within a strict 15-minute window. No internet assistance allowed.
2. Rules
  1. Time: 15 Minutes Hard Stop.
  2. Languages: C, C++, Java, Python, JavaScript.
  3. Banned: ChatGPT, GitHub Copilot, StackOverflow.
  4. Allowed: Local IDE documentation.
3. Scoring
  • Correctness (60%): Passing hidden test cases.
  • Speed (30%): Time of submission.
  • Style (10%): Code readability.
