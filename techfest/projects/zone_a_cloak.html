Project Details: Invisible Cloak
Zone: A (The Arena) ID: A-03
1. Overview
An "Optical Camouflage" experience. Users cover themselves with a specific colored cloth (Red/Green), and the system renders the background over those pixels, making them appear invisible.
2. Technical Implementation
  • Stack: Python (OpenCV).
  • Logic: HSV Color Masking -> Inversion -> Bitwise AND with Reference Background.
  • Hardware: Tripod-mounted Webcam (Must be stationary).
3. Requirements
  • Red or Green Cloth.
  • Controlled Lighting (No shadows).
  • Large Display for User Feedback.
